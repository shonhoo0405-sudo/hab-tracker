## 📦 GitHub Pages дээр байршуулах заавар

### 1. Repository үүсгэх

```
1. github.com → "New repository" дарна
2. Repository name: hab-tracker
3. Public сонгоно
4. "Create repository" дарна
```

### 2. Файлуудыг upload хийх

```
1. Repository дотор "uploading an existing file" дарна
2. index.html болон README.md файлуудыг чирж оруулна
3. "Commit changes" дарна
```

### 3. GitHub Pages асаах

```
1. Repository → Settings → Pages
2. Source: "Deploy from a branch"
3. Branch: main / (root)
4. Save дарна
5. Хэдэн минутын дараа: https://YOUR-USERNAME.github.io/hab-tracker/
```
